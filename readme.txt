Amazfit bip themes consists of set of numered .png files and .json main file
You can edit them in paint and notepad or use more convinience online editors like https://amazfitwatchfaces.com/editor/watchfaceEditor/?bip
To compile themes use command line tool from tools folder
Result of compillation is .bin file
To put theme to your watch use "Amazfit Bip / Lite & Cor WatchFaces" app from android market.
This app can replace original mifit app themes by your .bin file from phone file system.
Then set "original" theme to your watch in mifit app.
